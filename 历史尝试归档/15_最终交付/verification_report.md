## 🔍 验证过程详情

### 技术配置

- **API提供商**: OpenRouter
- **模型组合**: 
  - anthropic/claude-sonnet-4.5
  - openai/gpt-5
  - google/gemini-2.5-pro
- **并发处理**: 50个题目同时验证
- **每题验证次数**: 3次（每个模型1次）
- **总API调用**: 1,279题 × 3模型 = **3,837次调用**

### 验证流程

```
题目 → Claude 4.5 验证 ┐
    → GPT-5 验证      ├─→ 结果汇总 → 判断状态
    → Gemini 2.5 验证 ┘

判断逻辑:
- 所有3个模型都认为正确 + 高置信度 → approved
- 至少2个模型认为正确 → approved  
- 所有模型都认为错误 → rejected
- 其他情况 → needs_review
```
